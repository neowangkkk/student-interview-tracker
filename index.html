🔧 Cloud Setup Instructions

Step 1: Create a Google Sheet

• Go to sheets.google.com

• Create a new sheet named "Interview Tracker"

• Add headers: A1="Interview", B1="Student1", C1="Student2", D1="Student3", E1="Timestamp"

Step 2: Make it Public

• Click "Share" button

• Change to "Anyone with the link" can edit

• Copy the share link

Step 3: Update Code

• Replace SHEET_ID in the code with your sheet ID

• The sheet ID is the long string in your Google Sheets URL

Example:

URL: https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms/edit
Sheet ID: 1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms
